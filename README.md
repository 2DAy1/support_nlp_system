# Support NLP System

Система маршрутизації звернень служби підтримки з NLP-класифікацією.

## Опис

Проєкт реалізує веб-систему для автоматичної обробки звернень користувачів.  
Користувач створює звернення, після чого система аналізує текст за допомогою NLP-класифікатора на основі TF-IDF та Naive Bayes.

Після класифікації система автоматично визначає:

- категорію звернення;
- рівень впевненості класифікації;
- відповідальний відділ підтримки.

## Функціонал

- створення звернень;
- автоматична NLP-класифікація;
- автоматична маршрутизація до відділу;
- перегляд списку звернень;
- перегляд деталей звернення;
- зміна статусу звернення;
- історія зміни статусів;
- статистика за категоріями, відділами та статусами;
- адміністративна панель Django.

## Технології

- Django
